# Update Windows using Ansible
## Jenkins Job to Update windows VM with Ansible

### Hosts 
As Ansible looks at [/etc/ansible/hosts][df1] for inventory by default, [hosts][df1] file has the list of IP's that the playbook runs. Update the [hosts][df1] file incase of further addition of the nodes 

### Jenkinsfile

Jenkinsfile contains the tasks performed by jenkins in stages. 

- [Checkout][df1] - Clone the Repository
- [Update the ansible Inventory][df1] - Updating the /etc/ansible/hosts file
- [Run Ansible Playbook][df1] - Execute the playbook 

### win_updates.yaml
Ansible playbook to update windows
